Use the green buttons to confirm user actions. Use the red buttons to cancel an operation or changes. It is not recommended to stud a page with too many colored buttons. A samples on the left illustrates the buttons in unpressed and pressed states. Hover the mouse pointer over any unpressed button to see how it looks in hover state.

Text Input Controls

Colors

Main text on white background: #555555  
Extra text on white background: #999999  
Link color on white background: #1f67b0
Main text on dark gray background: #dbdfe7
Main text on gray background: - #e3ebed
Use light shades for backgrounds (see the first text line).

To highlight a text or an idea, use colors like in the third line. 

Use beige color (#fdf9e5) to display updates (new messages, comments, text blocks showing new information).

Don't underline links, but rather accent them with a different color instead.
Font
 The font to use on iOS is Helvetica. The font to use on Windows is Arial.

All font styles are possible.
Headings

First level heading - 30px

Second level heading - 14px (bold)

 For headings, use the font size specified in the example on the left.
